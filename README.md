# 🤖 AI Career Companion – Resume Analyzer & Career Advisor

An intelligent AI-powered career agent that analyzes resumes, calculates ATS score, job match percentage, identifies skill gaps, generates career roadmaps, simulates future growth, and provides ChatGPT-like career guidance using a local LLM (Ollama).

---

## 🚀 Features

- 📄 Resume parsing (PDF/DOCX)
- 📊 ATS score & resume quality scoring
- 🎯 Job role matching
- 🧠 Skill gap analysis
- 🗺️ Personalized skill roadmap generation
- 🔮 Career growth simulation
- 📝 Resume improvement suggestions
- 💬 AI CareerBot chat (ChatGPT-like)
- 👤 Login & forgot password system
- 🧾 AI-generated PDF reports
- 🧠 Agent personality + session memory

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Ollama (Mistral LLM)
- NLP (spaCy)
- PDF generation
- Local AI inference
